# MPGreen - Part 1: 초기 화면 구현 (HTML & CSS)

## 📌 프로젝트 개요 및 보관 사유 (Archived)
이 레포지토리는 최종 웹 애플리케이션 프로젝트인 'MPGreen'의 **초기 화면 구현 단계 이력**을 보존하기 위해 보관 처리되었습니다. 강사님께서 제공해주신 **화면설계(Figma)**을 기반으로 HTML 구조 및 CSS 스타일링을 연습하고 구현하는 데 집중했습니다.

* **프로젝트명:** BNK 국비 교육 과정 학습 과제 - 대학교 회원 및 관리자 홈페이지 구현 (Part 1)
* **핵심 목표:** 화면설계를 기반으로 한 **정적 화면 설계 및 퍼블리싱 연습**
* **활용 기술:** HTML5, CSS3 (화면 설계 기반의 크기, 색상 등 디자인 시스템 구현)
* **보관 처리 이유:** 해당 코드는 이후 **JSP 및 Java 백엔드 로직**과 통합되어 Part 2 레포지토리에서 최종 완성되었습니다.

## 💻 디자인 소스 및 구현 이력

### 1. Figma 디자인 소스

이 프로젝트는 아래 Figma 주소의 화면 설계에 따라 크기, 색상, 레이아웃을 100% 구현했습니다.

* **Figma 원본 주소:** [https://www.figma.com/design/XKYxlYEaCX1hiJ9ZJcoZRo/%EA%B7%B8%EB%A6%B0%EB%8C%80-%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8?node-id=0-1&p=f&t=tEgZ5yc1wMmU3X3y-0]

### 2. 주요 구현 내용

* **HTML 구조 설계:** 대학교 회원 및 관리자 페이지에 필요한 논리적 섹션 구조(Header, Navigation, Main Content, Footer) 구축.
* **CSS 스타일링:** Figma에서 제공된 색상 팔레트 및 크기(Spacing, Font Size)를 그대로 적용하여 디자인 시스템 구현.
