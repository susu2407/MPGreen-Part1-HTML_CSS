# MPGreen - Part 1: 초기 화면 구현 (HTML & CSS)

## 📌 프로젝트 개요 및 보관 사유 (Archived)
이 레포지토리는 최종 웹 애플리케이션 프로젝트인 'MPGreen'의 **초기 화면 구현 단계 이력**을 보존하기 위해 보관 처리되었습니다. 
강사님께서 제공해주신 **화면설계(Figma)**을 기반으로 HTML 구조 및 CSS 스타일링을 연습하고 팀원 간의 역할 분담 및 협업 룰을 정립하는데에 집중했습니다.

* **프로젝트명:** BNK 국비 교육 과정 학습 과제 - 대학교 회원 및 관리자 홈페이지 화면 구현 (Part 1)
* **핵심 목표:** 화면설계를 기반으로 한 **정적 화면 구현 연습**
* **활용 기술:** HTML5, CSS3 (화면 설계 기반의 크기, 색상 등 디자인 시스템 구현)
* **보관 처리 이유:** 해당 코드는 이후 **JSP 및 Java 백엔드 로직**과 통합되어 Part 2 레포지토리에서 최종 완성되었습니다.
